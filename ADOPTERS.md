# Adopters

Are you using **semantic-ui** in a project, product, or experiment? **We'd love to know.**

Listing yourself is **optional and free**. It helps the project (social
proof, prioritising features) and it helps you (a public reference that
you build on it).

## How to add yourself

Open a pull request that adds a row to the table below. Anything you're
comfortable sharing is fine — even just a name.

## Prefer not to open a PR?

Start a thread under **[Discussions](https://github.com/mindconnect-ai/mc-semantic-ui/discussions)**
(or open an issue). No email needed — that reaches the maintainer directly.

The project is free and Apache-2.0 licensed. If you'd like help integrating
